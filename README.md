## Installation

Install node dependencies in the command line via `npm i`  
MySQL is also required for this application to run.  
A `.env` file is required in the root folder, to hold the following code:

```code
DB_USER=username
DB_PASSWORD=pass
DB_DATABASE=mydb
```
